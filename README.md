# Chess Gym — webapp (Level 2/3)

Static, no-build webapp: play against the AI and work the Mastery Map (`gym`) in the browser,
with progress saved to `localStorage`.

**Not yet supported here** (CLI-only for now — see `docs/USER-GUIDE.md`): hotseat (two players,
one keyboard) and async play-by-mail matches. That means **Arena "Head to Head" (A4) milestones —
finishing or winning an async match — currently require the CLI**, not this webapp.
